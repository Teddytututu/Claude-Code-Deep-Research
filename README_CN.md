# Deep Researcher

> 多智能体深度研究系统，生成引用详实的综合性研究报告

## 能做什么

输入一个主题 → 获得 **Gemini Deep Research 风格的报告**：
- 学术论文 (ArXiv)
- GitHub 仓库
- 社区讨论 (Reddit, HN)
- 引用网络 & 研究缺口
- 双语输出 (中文叙述 + 英文术语)

## 快速开始

```
# 基础
深度研究 [主题]

# 指定时间
深度研究 [主题]，给我1小时
```

## 定制输出格式

```
# 博客文章
深度研究 [主题]，写一篇3000字的博客文章

# PPT 大纲
深度研究 [主题]，生成PPT大纲

# 代码示例
深度研究 [主题]，整理代码示例

# 对比表
深度研究 [主题]，做一个框架对比表

# 技术提案
深度研究 [主题]，写一份技术提案

# JSON 格式
深度研究 [主题]，输出JSON格式
```

## 输出内容

| 报告 | 字数 | 目标读者 |
|------|------|----------|
| **综合报告** | 6,000-8,000 字 | 工程师、决策者 |
| **文献综述** | 3,000-5,000 字 | 研究者、学者 |

## 核心特性

- **并行研究**：3 个代理同时搜索 ArXiv、GitHub、社区
- **质量保证**：独立评估 + 自动链接验证
- **定制输出**：博客文章、PPT、代码示例、对比表、技术提案、JSON
- **引用图谱**：Mermaid 图表展示关系

## 系统架构
